## Distributing Roles
### 👨‍💻 Front-end
* 팀원 강동협: 퍼블리싱, UI/UX 총괄, 회원가입, 로그인, 홈피드, 마이페이지, 플레이리스트, 좋아요
* 팀원 최진석: API 연결 관리, User, 마이페이지, 음원, 채팅, 검색, 예외처리




### 👨‍💻 Back-end
* 팀장 정혜윤 : CI/CD, User, 시큐리티, 플레이리스트, 팔로우, 좋아요 기능 구현
* 팀원 민광식 : API 연결, 검색, 음원, 채팅 기능 구현
